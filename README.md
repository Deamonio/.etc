
# 📂 Deamonio/.etc

이 레포지토리는 이름 그대로 **`.etc` (기타, 그 외)** 파일들을 모아두는 공간입니다. 개인적인 학습 자료, 미니 프로젝트, 스크립트, 설정 파일 등 다양한 종류의 파일들이 정리되어 있습니다.

---

## 📌 레포지토리 목적

* **지식 저장소:** 평소에 진행하거나 관심을 가졌던 다양한 분야의 코드 조각, 자료, 분석 결과 등을 한곳에 모아 보관합니다.
* **빠른 참조:** 필요할 때 언제든지 특정 코드나 자료를 쉽게 찾고 참조할 수 있도록 합니다.
* **진행 중인 프로젝트 기록:** 독립적인 레포지토리를 만들기엔 규모가 작은 실험적인 프로젝트나 학습 내용을 기록합니다.

---

## 💡 주요 프로젝트 및 내용

현재 레포지토리에 포함되어 있거나 계획 중인 주요 폴더 및 파일 목록입니다.

| 이름 | 내용 | 주요 언어/분야 |
| :--- | :--- | :--- |
| **`02. prevent water waste system`** | 물 낭비 방지 시스템 관련 프로젝트 파일 및 코드 | C++/임베디드 시스템 |
| **`아가모토(선배작품).ipynb`** | 선배의 작품 또는 참고 자료로 활용되는 Jupyter Notebook 파일 | Jupyter Notebook/데이터 분석 |
| **기타 스크립트** | 일상적인 작업을 자동화하거나 유틸리티 기능을 수행하는 작은 스크립트 | Python, Shell Script 등 |

## 🛠 기술 스택 (Tech Stack)

이 레포지토리 내의 프로젝트에서 주로 사용되는 기술 스택입니다.

* **Jupyter Notebook / Python:** 데이터 분석, 머신러닝 학습, 간단한 테스트 코드 작성에 주로 사용됩니다.
* **C++:** 임베디드 시스템 또는 특정 성능 요구 사항이 있는 프로젝트에서 사용됩니다.
* **Markdown:** 문서 작성 및 정리 (이 README 파일 포함)
